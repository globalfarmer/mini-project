To run project follow these steps bellow

1. To run server
   Open a terminal, move to 'server' folder then 
   run ./run.sh <port number>where <port number> is 
   the port you wish the server listen on

2. To run client open 
   Open another terminal, move to 'client' folder,
   then install npm, then install node packages
   by run `npm install` and then `npm start` to run

NOTE: Client is default call server on 8000 port
      if you want client call server on the others
      please change proxy property in client/package.json
